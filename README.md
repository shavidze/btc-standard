# ბიტკოინ სტანდარტი — ინტერაქტიული კურსი

ქართულენოვანი ინტერაქტიული სასწავლო კურსი საიფედინ ამმუსის „ბიტკოინ სტანდარტის" მიხედვით.

- 9 თავი · 57 გაკვეთილი · 72 საგამოცდო კითხვა
- ინფლაციის სიმულატორი
- პროგრესი ინახება ბრაუზერში (localStorage)

## სტრუქტურა

- `index.html` — მთელი აპლიკაცია (სტატიკური, dependency-ების გარეშე)
- `content/ch*.json` — თავების გაკვეთილები და ქვიზები
- `content.js` — აწყობილი კონტენტი (`python assemble.py` აგენერირებს)
- `build_content.py` — კონტენტის გენერაცია Gemini API-თ (key: `~/.gemini_api_key`)

საგანმანათლებლო მიზნებისთვის.
